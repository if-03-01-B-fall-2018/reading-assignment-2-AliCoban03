3.Specify colors by color names and in hexadezimal notation

In HTML, a color cam be specified using a hexadecimal value in the form#rrggbb. Where rr (red), gg(green) and bb (blue) are hexadecimal values between 00 and ff (same as decimal 0-255). For Example, #ff0000 is displayed as red because red is set to its highest value (ff) and others are set to lowest value (00).

#ff0000

DogderBlue

Black

Gray

Light Gray

White

#ffffff

5.Pseudo classes for the < a > element

/* unvisited link */

a:link {

color: #ff0000

}



/* visited link */

a:visited {

color: #00ff00;

}



/* mouse over link */

a.hover {

color: #ff00ff;

}



/* selected link */

a.active {

color: #0000ff

}

10.Identify the difference between serif and sans-serif fonts

"Sans" means without serif - these fonts do not have the lines at the ends of characters

"Serif": A Serif font has “feet” or added extensions like the top and bottom of the Times New Roman serif font example in the illustration.
