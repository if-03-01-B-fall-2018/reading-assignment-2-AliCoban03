Web Server

1. What is a web server?

Awnser: A web server is server software, or hardware dedicated to running said software, that can satisfy World Wide Web client requests.

2. What does a web server?

Awnser: It stores, process and deliver web pages to clients.

3. How can content be stored on a web server?

Awnser: On the Hypertext Transfer Protocol (HTTP)

4. How is content in webpages described?

With a HTML document.
