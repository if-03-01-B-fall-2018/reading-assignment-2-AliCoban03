Web Client

1. What is a web client?

Awnser: It is a client-side component within the Java 2 Platform Enterprise Edition (J2EE) and it's a distributed multi-tiered application model used for building and developing enterprise applications

2. What does a web client?

Awnser: It performs client-side operations as they might need acces to information available only on the client side, like user input, or beacause the server lacks the processing power necessary in such operations

3. Web Client and Web Browser is the same??

Awnser: No, the main difference between a Web browser and a Web server is that Web browser requests for the document and services, an act as an interface between a client and a server which displays the web content. On the other hand, Web server accepts, approve, and response to the request made by a web browser for a web document
